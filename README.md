<h1>Projeto Integrador - Beauty Salon</h1>

<h2>Descrição do Projeto</h2>
<p>A ideia do projeto é dar continuidade a uma atividade que estavámos desenvolvendo na disciplina de Projeto Orientado a Objetos
do 2º Semestre do curso de Ciência da Computação pela FAESA. Essa atividade consistia em criar um agendamentos de cortes de cabelo de forma virtual, para ajudar
barbeiros a alavancar seu negócio e atingir mais clientes ao redor do seu local de trabalho.</p>
<p>Diante disso, esse projeto está em fase de desenvolvimento e esperamos concluir, pelo menos, a parte inicial que consiste do usuário fazer um agendamento 
com o salão, sem as outras funcionalidades especiais, que vamos explorar com o decorrer do tempo.</p>

<p>Por fim, o nosso grupo é composto por 6 pessoas, contando comigo. Estamos todos trabalhando e aprendendo em conjunto para que o projeto seja realizado. Vale ressaltar,
que apenas 1 integrante do grupo tem uma experiência maior com outras linguagens, portanto todo o restante está aprendendo com o desenrolar da programação do trabalho</p>
